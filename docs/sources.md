# Source texts — canonical URLs, retrieval, licences

All texts used by the project are either public domain (US) or
under a permissive licence with attribution. Treat this file as the
single authoritative record of *where* each text came from and
*under what terms* the project redistributes it.

## Headline pair — Greek NT vs KJV

### SBLGNT (Greek New Testament, Holmes 2010)

* **Repository:** <https://github.com/morphgnt/sblgnt>
* **Files used:** `61-Mt-morphgnt.txt` through `87-Re-morphgnt.txt`
  (27 NT books, morphologically tagged, one word per line)
* **Edition:** Michael W. Holmes (ed.), *The Greek New Testament:
  SBL Edition*, Society of Biblical Literature & Logos Bible
  Software, 2010.
* **Licence — text:** SBLGNT EULA / CC BY 4.0 (see
  <http://sblgnt.com/license/>). Attribution required.
* **Licence — morphology:** CC BY-SA 3.0 by James Tauber et al.
* **Our use:** we read the morphgnt files to reconstruct the
  *running Greek text* (column 4 of each line, concatenated within
  a verse). We do not redistribute the morphological parse; we use
  it only to recover verse text.
* **Citation block (paste into the notebook / post):**
  > Michael W. Holmes (ed.), *The Greek New Testament: SBL Edition*
  > (Society of Biblical Literature & Logos Bible Software, 2010).
  > Text used under the SBLGNT licence (CC BY 4.0).
  > Morphology by J. K. Tauber (ed.), *MorphGNT: SBLGNT Edition*,
  > v6.12, CC BY-SA 3.0. <https://github.com/morphgnt/sblgnt>

### KJV (King James Version, 1769 Cambridge)

* **Repository:** <https://github.com/aruljohn/Bible-kjv>
* **Files used:** per-book JSON (`Matthew.json` … `Revelation.json`),
  plus `Books.json` for the canon order.
* **Edition:** the standard 1769 Cambridge text (the spelling
  modernisation of the 1611 first edition; this is what every
  modern KJV print reflects).
* **Licence:** public domain in the United States; Crown copyright
  in the United Kingdom (the project does not redistribute within
  the UK; for an authoritative public-domain English Bible inside
  the UK use the WEB or ASV).
* **Citation block:**
  > The King James Version of the Bible, 1769 Cambridge edition
  > (public domain). Verse-keyed JSON via the `aruljohn/Bible-kjv`
  > GitHub repository, retrieved 2026-05-17.

## Project Gutenberg redistribution

Project Gutenberg permits unlimited non-commercial redistribution
of its etexts when the header is preserved; for commercial use the
PG licence must be reviewed. This project keeps the raw downloads
in `data/raw/` (git-ignored) and only commits *derived* tidy JSON.
The PG header / footer is stripped during ingest and not
redistributed.
